# IT Support Agent Demo

A demo IT support chatbot that handles tickets, executes shell scripts, and maintains conversation context.

## Setup

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
   3. Copy `.env.example` to `.env` and add your OpenAI API key:
   cp .env.example .env
   4. Run the agent:
   python demo/main.py
